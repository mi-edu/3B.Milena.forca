#  Jogo da Forca em Python

---


##  Funcionalidades

 **Palavras Aleatórias:** Seleção automatizada de palavras técnicas do ecossistema de programação e tecnologia.
 **Gráficos em ASCII:** O boneco da forca é desenhado no terminal em tempo real conforme você erra as letras.
 **Sistema de Pontuação e Recorde:** * Ganhe **+10 pontos** a cada letra acertada. Perca **-2 pontos** a cada tentativa errada.
  O jogo guarda o seu **Recorde** enquanto a sessão estiver aberta!
 **Validações Inteligentes:** O sistema impede que você digite números, caracteres especiais, múltiplos caracteres ou letras que já foram tentadas antes.
 **Loop de Jogo:** Opção de jogar novamente ao final de cada partida sem precisar reiniciar o script.

---
## Difereça
O diferencial do código original é que eu adicionei uma **forca**, coloquei um novo **loop** para jogar novamente e também coloquei um tema de **recorde**
