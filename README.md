
#  3B.Milena.forca

## Jogo da Forca em Python

---

#  Funcionalidades

*  **Palavras Aleatórias**
  Seleção automática de palavras técnicas do ecossistema de programação e tecnologia.

*  **Gráficos em ASCII**
  O boneco da forca é desenhado no terminal em tempo real conforme os erros acontecem.

*  **Sistema de Pontuação e Recorde**

  * Ganhe **+10 pontos** a cada letra correta
  * Perca **-2 pontos** a cada tentativa errada
  * O jogo guarda o seu **recorde da sessão** enquanto o programa estiver aberto

*  **Registro de Letras Erradas**
  Todas as letras incorretas digitadas são registradas e exibidas na tela, ajudando o jogador a acompanhar os erros e melhorar a estratégia.

*  **Loop de Jogo**
  Permite jogar novamente ao final de cada partida sem precisar reiniciar o script.

---

#  Diferenciais do Projeto

O diferencial desta versão do jogo da forca é a adição de:

*  **Forca visual em ASCII**
*  **Sistema de replay (jogar novamente)**
*  **Sistema de recorde da sessão**
*  **Exibição de letras erradas na tela**

Esses elementos tornam o jogo mais interativo, organizado e divertido para o usuário.
